# Pet Care Knowledge Base

This folder contains documents that will be processed into the FAISS vector store for the AI chatbot.

## Supported Formats
- PDF files (.pdf)
- Text files (.txt)

## Adding Knowledge
1. Add your pet care documents (PDFs or text files) to this folder
2. Run `python create_vector_db.py` to process them
3. The vectorstore will be created in the `vectorstore` directory

## Sample Content
A sample pet care guide has been created automatically. You can add more comprehensive documents for better AI responses.
